Welcome to the Archive! This page is dedicated to the archiving of old versions of the Fossils and Archaeology mod.

The Fossils and Archaeology mod has existed since beta 1.7.2, however only some versions are still available.

This list has been collected by Willatendo, Skidmark, and a now-deleted user.

In order to use really old versions, you will need to use a third-party launcher as Modloader no longer works on the standard launcher.

This comes with a general warning: modifying the game for these old versions can cause major issues. I find it is ALWAYS best to use a third-party launcher when messing with old versions. I use Multimc however any work!

The version I MOST RECOMMEND is for 1.2.5, 6.8 Patch 1. I played a ~30 episode play-through with this version and I found it the most stable. There are issues - many specific features don't really work - however most things do and I don't think I ever had any crashes with it. Your mileage may vary.

I do not own nor did I make these files, they are all owned by Flammarilva/Team July and hosted here. Most non-F/A links are from MCArchive so go check them out!

You need to go find Java 8 to play these versions, I think mine comes from Adoptium

Beta 1.7.3

Download 4.3

Download 4.8

Key Features: Triceratops, Velociraptor, and Tyrannosaurus

Requires: Java 8, Modloader, and Audio Mod

Installation Guide:

  • Ensure you have Java 8 installed and Multimc or equivalent third-party launcher.
  • Create a new instance of Beta 1.7.3.
  • Go to settings and change the Java to your Java 8 installation.
  • Run it to ensure it works.
  • Close the game and go to version.
  • Once in version, click "add to Minecraft.jar" and select ModLoader, AudioMod, and mod_Fossil.
  • Run the game and create a test world to ensure it has installed. To be sure, since there is no creative mode, you can also install in the same way HowManyItems.

Sounds

Download

Not all versions come bundled with sounds due to copywrite issues. This is the pack of sounds. They are installed differently!

Installation Guide:

  • Install your desired version and check if sounds are included (This definetly is needed for 1.2.5 and 1.3.2!).
  • Go to the .minecraft folder and navigate through /resources/mod/sound/ folders.
  • Unzip your sounds, and put the files in it into the /sound/ folder.
  • Run the game and you will have sounds!
